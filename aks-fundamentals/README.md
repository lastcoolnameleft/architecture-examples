# AKS Fundamentals

> Quick-reference commands and walkthrough for getting started with AKS.

## Contents

| File | Description |
|------|-------------|

## When to Use This

- You're onboarding to AKS and want a structured walkthrough
- You need quick copy-paste kubectl commands for common tasks
- You're running a demo or workshop and need a cheat sheet

## Prerequisites

- Azure CLI installed and authenticated
- `kubectl` installed
- An Azure subscription with permissions to create AKS clusters
